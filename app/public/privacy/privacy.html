eduMeet – Privacy Notice

 

What is eduMeet?

eduMeet is a easy and cost-effective video-conferencing service for research, educational and art societies that enables real-time communication over peer-topeer connections. eduMeet is an Open source software, created by research society for research society.

 

How it works?

eduMeet is simplifing real-time communication by introducing web browser based tools. In order to use eduMeet it is needed to run eduMeet in your browser – there is no need to install clients or plugins.

 

Why We Process Personal Data?

If you wish to use eduMeet service the only thing you need to do is to open a browser and enter into eduMeet room to start your videoconferencing. eduMeet processes a small amount of end user personal data for security and statistical purposes.

 

What Personal Data We Process?

During this process to provide you eduMeet service we will process:

a. IP address (every type of discoverable IP, as host, NAT addresses, etc.)

b. client browser (type, version, codecs available)

In case of authenticated users we process also:

a. OpenID sub ID

b. Profile picture

c. Name

d. E-mail

 

What is stored and for how long?

That depends on how long a given installation chooses to keep the logs, if at all. The data during runtime for a guest user is deleted as soon as the user disconnects.

For authenticated users:

a. OpenID

b. Profile picture

c. Name

d. E-mail

is kept for the same duration as the token expire time from OpenID or the user logs out.
